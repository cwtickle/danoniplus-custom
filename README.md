# Dancing☆Onigiri(CW Edition) - customjs
[Dancing☆Onigiri(CW Edition)](https://github.com/cwtickle/danoniplus) の自作品用customjs

自作品用のjsを管理するためのものです。  
あくまで自分用なので他の用途には使いにくいと思いますが、参考に公開しておきます。
